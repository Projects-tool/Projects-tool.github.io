供应商智能采集系统

项目简介

一款面向采购人员的多平台供应商搜索采集工具。通过内嵌浏览器自动化技术，实现在 1688、淘宝、京东、拼多多、百度爱采购等主流电商平台上一键搜索并采集供应商信息，大幅提升供应商筛选效率。

技术栈

C# WPF .NET 8 WebView2 Edge Chromium EPPlus

核心功能

  • 多平台采集 — 支持 1688、淘宝、京东、拼多多、百度爱采购等平台，统一搜索入口
  • 智能去重 — 自动识别重复供应商，按名称、地址、联系方式多维度去重
  • 评分算法 — 内置供应商质量评分模型，综合考量信誉、交易量、响应速度等指标
  • 品牌溯源 — 自动鉴别品牌授权链路，辅助判断供应商真实性
  • Excel 导出 — 采集结果一键导出为格式化的 Excel 文件，方便后续分析

技术架构

基于 WPF + WebView2 构建，利用 Edge Chromium 内核实现网页自动化操作。前端界面采用 MVVM 模式，数据层通过 JavaScript 注入与网页交互，采集结果经过清洗、去重、评分后存入内存数据集，最终通过 EPPlus 库导出 Excel。

使用方式

下载后解压，双击 供应商采集系统.exe 即可运行。在搜索框输入关键词，选择目标平台，点击搜索即可开始采集。采集完成后点击导出按钮保存结果。

AI 提示词优化系统

项目简介

一款原生 WPF 桌面应用，输入任意提示词即可一键优化为结构化高质量 Prompt。内置规则引擎覆盖 23 个专业领域，完全本地运行，无需联网，无需 API Key。

技术栈

C# WPF .NET 8 规则引擎 LCS Diff

核心功能

  • 三种优化模式 — 简洁模式快速补全、详细模式全面扩展、专业模式顶级结构化
  • 23 领域角色推断 — 自动识别编程、写作、翻译、设计、金融、医疗、游戏等领域并匹配专家角色
  • 场景化思维链 — 根据任务类型生成定制化的思考框架
  • 智能分段 — 长文本自动拆分为子任务，避免结构混乱
  • 语气清理 — 口语化表达自动转换为精确指令
  • 中英文自适应 — 自动检测语言并生成对应语言的优化结果
  • Diff 对比 — 基于 LCS 算法的红绿标注差异对比视图
  • 六维雷达图 — 角色、任务、约束、格式、上下文、具体性六个维度可视化评分
  • 多轮迭代 — 支持对优化结果再次优化，逐步打磨
  • 历史记录 — JSON 持久化存储，关闭后重开历史还在
  • 亮暗主题 — 一键切换亮色/暗色主题
  • 10 个预设模板 — 代码审查、文案撰写、翻译润色、数据分析等常见场景

技术架构

纯 WPF 原生应用，不依赖任何 Web 技术或外部 API。核心是一个基于规则的 Prompt 优化引擎，通过关键词匹配、结构分析、模板填充等策略对输入文本进行结构化重组。评分系统从语义密度、指令清晰度、结构完整度、具体性、模糊词惩罚等多个维度综合打分。Diff 引擎基于最长公共子序列算法实现逐行对比。

优化输出结构（专业模式）

  • System Prompt — 精准角色画像 + 专业能力描述
  • 上下文 — 结构化场景描述
  • 核心指令 — 智能分段的子任务列表
  • 约束条件 — 领域针对性约束
  • 避免事项 — 负面约束边界
  • 受众与语气 — 自动推断受众级别
  • 思考方式 — 场景化思维链
  • 输出规范 — 格式要求

使用方式

下载后解压，双击 AI提示词优化系统.exe 即可运行。在输入框输入提示词，选择优化模式，点击"开始优化"。支持快捷键 Ctrl+Enter 优化、Ctrl+Shift+C 复制、Esc 清空。

ReportBot 智能报表

项目简介

一款智能数据分析与报表生成工具。导入 Excel 或 CSV 文件后，自动识别数据列类型，智能选择合适的图表类型进行可视化，支持多种格式导出和 IM 消息推送。

技术栈

C# WPF .NET 8 ScottPlot ClosedXML CsvHelper

核心功能

  • 智能列识别 — 自动判断日期列、数值列、分类列，无需手动配置
  • 趋势分析 — 时间序列数据自动生成折线图，展示变化趋势
  • 维度对比 — 分类数据自动生成柱状图、饼图，直观对比各维度
  • 同比分析 — 支持按年/月/周进行同比环比分析
  • 多格式导出 — 一键导出为 Excel、HTML、PDF 格式的报表
  • IM 推送 — 支持飞书、钉钉、企业微信 Webhook 和邮件推送报表
  • 定时任务 — 内置定时器，可设置周期性自动生成并推送报表
  • 报表模板 — 预设多种报表模板，快速生成标准化报告

技术架构

基于 WPF 构建界面，使用 ClosedXML 读取 Excel、CsvHelper 解析 CSV 数据。数据分析层自动推断列类型并选择最佳可视化方案。图表渲染使用 ScottPlot 库，支持折线图、柱状图、饼图、散点图等多种类型。导出模块支持 Excel（带嵌入图表）、HTML（响应式网页报表）、PDF 三种格式。

使用方式

下载安装包后运行安装，启动 ReportBot。点击"导入数据"选择 Excel 或 CSV 文件，系统自动分析并生成图表预览。调整参数后点击"导出报表"保存，或配置 Webhook 地址实现自动推送。
